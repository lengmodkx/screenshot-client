# 截图客户端界面重新设计方案

## 需求背景

重新设计截图客户端的界面，采用卡片式布局，提升用户体验。

## 设计决策

1. **布局风格**: 卡片式布局，4个独立卡片区块
2. **颜色主题**: 墨绿色 (#2d5a3d 渐变)，适合教育场景
3. **预览区大小**: 小缩略图 (80px高度)，节省空间

## 界面结构

```
┌─────────────────────────┐
│  截图客户端      [在线] │  <- 顶部导航 (墨绿色)
├─────────────────────────┤
│ [运行状态卡片]          │
│  当前状态: 工作中       │
│  存储模式: 云端上传    │
│  截图间隔: 10秒        │
│  登录用户: admin       │
├─────────────────────────┤
│ [今日统计卡片]          │
│   42    │   14:32:05  │
│ 今日截图 │   最后截图  │
├─────────────────────────┤
│ [最新截图卡片]          │
│  [缩略图预览]          │
│  已上传 - 14:32:05    │
├─────────────────────────┤
│ [控制面板卡片]          │
│  [■ 停止]  [📷 截图]  │
│  [设置]   [登出]       │
└─────────────────────────┘
```

## 组件说明

| 卡片 | 内容 |
|------|------|
| 运行状态 | 当前状态、存储模式、截图间隔、登录用户 |
| 今日统计 | 今日截图数量、最后截图时间 |
| 最新截图 | 缩略图预览、状态消息 |
| 控制面板 | 开始/停止、立即截图、设置、登出 |

## 配色方案

- 主色: #2d5a3d (墨绿色)
- 渐变: linear-gradient(135deg, #2d5a3d 0%, #3d7a52 100%)
- 卡片背景: #ffffff
- 卡片头部背景: #e8f0e8 (淡绿色)
- 状态指示: 在线 #52c41a / 离线 #ff4d4f
